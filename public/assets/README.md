# Novax Digital Assets

Lege finale Medien bitte in diese Ordner:

```txt
public/assets/logos/
public/assets/images/
public/assets/siegel/
public/assets/optimized/
```

Empfohlene Dateinamen:

```txt
public/assets/logos/novax-logo.svg
public/assets/logos/erfolg-buddies.svg
public/assets/logos/deutsche-pflegeentwicklung.svg
public/assets/logos/schulung-immobilienmakler.svg
public/assets/logos/passflow.svg

public/assets/images/team.jpg
public/assets/images/office.jpg
public/assets/images/hero-abstract.webp
```

Logos idealerweise als `svg` oder transparente `png`; Fotos als `jpg`, `png` oder `webp`.

`public/assets/optimized/` enthaelt kleinere Web-Derivate der Originalfotos. Diese Dateien werden aktuell in der Website verwendet, damit die Originalfotos als Master-Dateien erhalten bleiben, aber die Seite nicht unnoetig schwer wird.
